# redis_cache
Cálculo de fatorial usando node.js e redis.


Padrão de chave: "fatorial:num", sendo num o número para cálculo.

Rodar o arquivo fatorial.js e usar http://localhost:8081/fatorial/num substituindo num pelo número que desejar.

Exemplo: para cálculo de 6! usar http://localhost:8081/fatorial/6
